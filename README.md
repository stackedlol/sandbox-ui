# sandbox-ui

Glassmorphism component library for Next.js.

## Quick Start

```bash
npx create-sandbox-ui my-app
cd my-app
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Add Components to an Existing Project

```bash
npx sandbox-ui init
npx sandbox-ui add button card pricing
```

## Available Components

| Component      | Description                                      |
|----------------|--------------------------------------------------|
| `button`       | Primary, secondary, and ghost variants           |
| `card`         | 3D tilt card with cursor-following specular glow |
| `cryptic-text` | Animated text scramble/reveal effect             |
| `navbar`       | 3D carousel navbar with perspective transforms   |
| `pricing`      | Tiered pricing cards with premium glass effects  |
